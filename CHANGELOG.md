## [0.9.6] - 2026-01-14

### ✨ 新增功能
- **技能自动更新功能** - 实现了完整的技能更新系统
  - 后端版本追踪，支持检查技能更新
  - 集成 GitHub Commit API，基于文件变更检测更新
  - 两阶段更新流程（准备更新 → 确认更新）
  - 本地文件修改冲突检测
  - 原子化替换安装目录，带备份恢复机制
  - 更新前安全扫描
- **前端更新界面** - 已安装技能页面
  - 新增"检查更新"按钮
  - 技能卡片显示更新可用徽章和更新按钮
  - UpdateConfirmDialog 组件显示安全扫描报告
  - 冲突文件警告和强制覆盖选项
- **英文版 README** - 新增英文版项目说明文档

### 🔧 优化改进
- **状态显示优化** - 优化更新状态提示信息显示
- **首页 UI 微调** - 简化卡片组件代码结构

- 修复一些已知问题
